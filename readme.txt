A simple plugin to help you set up Quizzes behind a WooCommerce PayWall.

== Description ==

Quizzo is the perfect tool for setting up a simple Quiz where users can answer using a WC PayWall. Here is a list of its features:

- Easy set up for Quizzes, Questions and Answers.
- Enable Quiz purchase via WooCommcerce Paywall.
- Score module capturing for each Quiz session.
- Step-by-Step page feature for Quiz question.
- Timer/Counter feature for each Quiz question.
- Meta Data capturing for Quizzes, Questions and Answers.
